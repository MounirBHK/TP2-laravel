Création d'un site web dynamique en utilisant le cadriciel Laravel

Le mandat est de créer un site Internet pour recueillir de l'information auprès des étudiants du Collège Maisonneuve, et possiblement à l'avenir, de construire un réseau social pour eux.

  1. Nouveau projet Laravel : composer create-project --prefer-dist laravel/laravel MaisonneuveE2195646 "8.*"
  2. Créer la base de données : CREATE DATABASE TP1_Laraval_S4
  3. Modification du fichier .env : 
                            APP_NAME="MaisonneuveE2195646"
                        ...
                
                        DB_DATABASE=TP1_Laravel_S4
                        DB_USERNAME=root
                        DB_PASSWORD=
  4. Demarrage de Artisan : php artisan serve
  5. créer les modèles : 
                            php artisan make:model ville 
                        php artisan make:model etudiant
  6. Créer les tables (migrations) : 
                            php artisan make:migration create_villes_table
                        php artisan make:migration create_etudiants_table
                        php artisan migrate
  7. Saisir 15 nouveaux villes : php artisan make:factory VilleFactory 
                            public function definition()
                        {
                            return [
                                'nom' => $this->faker->city()
                            ];
                        }
  8. Création des données : 
                            php artisan tinker
                        \App\Models\ville::factory()->times(15)->create();
  9. Créer les contrôleurs : 
                            php artisan make:controller VilleController -m Ville
                        php artisan make:controller EtudiantController -m Etudiant
